# **App Name**: Module Hub

## Core Features:

- Module Dashboard: Display all the modules with their basic descriptions.
- Module Creation: Enable users to create new modules with title and content, and categorize these new modules with topic tags.
- Module Organization: Enable module reorganization; display order can be adjusted through drag and drop interface.
- Topic Generation: Generate potential topic tags for a new module by processing its title and content with a generative AI tool.
- Module Content Validation: Leverage a generative AI tool to ensure the newly written content doesn't clash in tone or contradict in information to previously added modules based on the title and content.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5) for a professional and reliable feel.
- Background color: Light gray (#F5F5F5) to provide a neutral backdrop.
- Accent color: Teal (#009688) to highlight interactive elements and key actions.
- Font: 'Inter', a grotesque-style sans-serif for both headlines and body, for a clean and modern aesthetic.
- Use a consistent set of minimalist icons for module actions.
- Employ a grid-based layout for organized display of modules.
- Use subtle transitions and animations for interactive elements, like the drag-and-drop module reorganization.
